Creating recipes¶

To probe a pair of topology and trajectory files of the bug atom group, you can use the polyphys.probe.prober.probe_bug() function:

polyphys.probe.prober.probe_bug(topology, trajectory, geometry, lineage, save_to='./', continuous=False)¶

runs various analyses on a lineage simulation of a ‘bug’ atom group in the geometry of interest.

Parameters

  • toplogy (str) – Name of the topology file.

  • trajectory (str) – Name of the trajectory file.

  • geometry ({'biaxial', 'slit', 'box'}) – Shape of the simulation box.

  • lineage ({'segment', 'whole'}) – Type of the input file.

  • save_to (str) – The absolute/relative path of a directory to which outputs are saved.

  • continuous (bool, default False) – Whether a trajectory file is a part of a sequence of trajectory segments or not.
